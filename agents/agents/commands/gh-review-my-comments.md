name: gh-review-my-comments
---

I added some comments in the PR. Use gh (GitHub CLI) to read the unresolved ones and fix the code or answer the comment.


## Rules

- Always add a robot (🤖 ) emoji in the begining of your comment.
- Do NOT resolve the threads.


## Emojis

When the comment start with a emoji, these are the meanings:

- Question (❓): just a question; don't change anything; just answer the comment in the same thread.
- Warning (⚠️): may be a problem. Read the comment and decide if it should be changed or just answered with a comment in the same thread.
- No Entry Sign (🚫): this is a blocker for the PR; fix it.
